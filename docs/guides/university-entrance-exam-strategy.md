## 💡 現在のコンテンツを大学受験向けに再評価

### 1. 英単語学習 → ✅ **そのまま活用・強化**

#### 現状評価
- ✅ **大学受験に必要不可欠**
- ✅ SRSで効率的に暗記
- ✅ 頻出単語データベース

#### 改善案

**a) 大学受験頻出度の可視化**
```
Q1/10: "abandon" (頻出度: ⭐⭐⭐⭐⭐)
意味: ||捨てる、見捨てる||
→ どの大学でよく出るかを表示
→ 「早稲田・慶應頻出」「共通テスト頻出」など
```

**b) 多義語の重点学習**
```
"abandon" には3つの意味があります：
1. 捨てる、見捨てる
2. あきらめる
3. 暴走する（abandon oneself to）
→ 全て覚えましょう！
```

**c) レベル別単語集**
- 基礎レベル（共通テスト）
- 標準レベル（中堅私大）
- 難関レベル（早慶・国公立）

**d) 過去問データ連携**
- 「この単語は〇〇大学の過去問で出ました」
- 実際の問題文も見れる

---

### 2. SVOCM（英文解釈）→ ✅ **大学受験の核、もっと強化**

#### 現状評価
- ✅ **難関大で必須**
- ⚠️ 専門用語でハードルが高い
- ⚠️ 単調で楽しさが薄い

#### 改善案

**a) 段階的学習システム**
```
Level 1: SとVだけ分けられる（基礎）
Level 2: SVOが分かる（標準）
Level 3: SVOCM全て分解（難関大レベル）
```

**b) 構文パターンの学習**
```
「it is ... that ...」構文を学びましょう
例文: "It is important that you study."
→ S: It / V: is / C: important / 真主語: that you study
```

**c) 過去問との連携**
```
この文型は「〇〇大学2023年 第2問」で出題されました
実際の問題を見てみましょう
```

**d) 解説の充実**
```
❌ 現在: "S: I, V: gave"
✅ 改善: "S: I (主語「私が」), V: gave (動詞「与えた」)"
     → なぜこれが正解なのかを説明
```

**e) ゲーミフィケーション**
- 正解すると「構文マスター」バッジ
- 難しい文型を解けるとレベルアップ
- ストリーク（連続正解）で報酬

---

### 3. 長文読解 → ✅ **大学受験の肝、時間制限重視に**

#### 現状評価
- ✅ **大学受験で最重要**
- ⚠️ 時間がかかりすぎる
- ⚠️ 実際の試験形式と異なる

#### 改善案

**a) 時間制限モード**
```
「早稲田過去問モード」
制限時間: 5分
長文読解 + 2問
→ 実際の試験形式に近づける
```

**b) レベル別問題**
- 共通テストレベル（基礎）
- 中堅私大レベル（標準）
- 難関大レベル（難関）

**c) 過去問データベース**
```
「早稲田大学 2023年 文学部」
→ 実際の過去問をDiscordで解ける
→ 解説も充実
```

**d) 速読トレーニング**
```
「1分でこの段落の要点を掴んでください」
→ 時間を意識した練習
```

**e) 段階的読解**
```
Step 1: パッセージを読む
Step 2: 設問を読む
Step 3: 選択肢を選ぶ
→ 各ステップでフィードバック
```

## 🔄 学習フローの改善（大学受験向け）

### 改善前
```
メニュー → 機能選択 → 問題 → 回答 → 終了
```

### 改善後

#### 1. オンボーディング
```
初回:
→ 志望校を設定
→ 現在のレベルを診断（簡単なテスト）
→ 学習目標を設定（例：「毎日30分」）
```

#### 2. 日常学習フロー
```
開始:
→ 今日の目標を表示（例：「早稲田頻出単語10個」）
→ 前回の学習状況（例：「昨日は長文読解で正答率80%」）

学習:
→ 志望校に合わせた問題が出題
→ 過去問形式で実戦練習

終了:
→ 結果サマリー（正答率、かかった時間）
→ 弱点分析（「関係代名詞が苦手です」）
→ 復習推奨（「間違えた単語を復習しませんか？」）
→ 明日の学習計画を提案
```
