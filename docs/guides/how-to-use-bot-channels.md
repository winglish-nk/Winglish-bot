# Botの使い方：どのチャンネルで使うか

> Winglish Botを使うためのチャンネルの説明

---

## 📍 Botを使えるチャンネル

### 1. **個人チャンネル（推奨）**

Botは自動的に**個人用の鍵チャンネル**を作成します。

#### チャンネルの場所
- **カテゴリ**: `Winglish｜個人学習`
- **チャンネル名**: `winglish-{あなたのユーザー名}`

例: `winglish-naruhito` のような名前のチャンネル

#### 個人チャンネルが作成されるタイミング

1. **Discordサーバーに参加した時**
   - 自動的に個人チャンネルが作成される
   - チャンネルにはあなたとBotだけがアクセスできる（鍵チャンネル）

2. **既に存在する場合**
   - 既存のチャンネルがそのまま使われる
   - 新しいチャンネルは作成されない

#### 個人チャンネルの確認方法

1. Discordサーバーの左側のチャンネルリストを確認
2. 「**Winglish｜個人学習**」カテゴリを探す
3. 自分のユーザー名がついたチャンネル（`winglish-{あなたの名前}`）があるか確認

---

### 2. **どのチャンネルからでも実行可能（スラッシュコマンド）**

スラッシュコマンド（`/notebook_create` など）は、**どのチャンネルからでも実行できます**。

- ✅ 個人チャンネルから実行可能
- ✅ 一般チャンネルから実行可能
- ✅ プライベートチャンネルから実行可能

---

## 🚀 使い方

### 方法1: 個人チャンネルで使う（推奨）

#### ステップ1: 個人チャンネルを開く

1. Discordサーバーの左側で「**Winglish｜個人学習**」カテゴリを探す
2. `winglish-{あなたのユーザー名}` というチャンネルをクリック
3. そのチャンネルで作業を開始

#### ステップ2: メニューを表示（オプション）

```
/start
```

このコマンドを実行すると、個人チャンネルにメインメニューが送られます。

#### ステップ3: 単語帳機能を使う

個人チャンネルで以下のコマンドを実行：

```
/notebook_create name:"テスト単語帳" description:"テスト用"
/notebook_list
/notebook_add notebook_name:"テスト単語帳" word:abandon
/notebook_study notebook_name:"テスト単語帳"
```

---

### 方法2: どのチャンネルからでも実行

スラッシュコマンドは、どのチャンネルからでも実行できます。

#### 例: 一般チャンネルから実行

1. サーバー内の任意のチャンネル（例: `#general`）を開く
2. スラッシュコマンドを実行：

```
/notebook_list
```

これでも正常に動作します。

---

## 🔍 個人チャンネルが見つからない場合

### 個人チャンネルが存在しない

**原因:**
- まだDiscordサーバーに参加していない
- Botがサーバーに追加されていない
- 個人チャンネルが作成されなかった

**解決方法:**

#### 方法1: `/start` コマンドを実行

任意のチャンネルで：

```
/start
```

このコマンドで個人チャンネルが作成され、そこにメニューが送られます。

#### 方法2: 管理者に依頼

管理者に以下のコマンドを実行してもらう：

```
/admin create_channel user:@あなたのユーザー名
```

これで個人チャンネルが作成されます。

---

## 💡 推奨される使い方

### 個人チャンネルで使うのがベスト

**理由:**
1. ✅ **プライバシー**: あなただけが見られる
2. ✅ **整理しやすい**: 学習履歴が一箇所にまとまる
3. ✅ **他の人に迷惑をかけない**: 一般チャンネルを埋め尽くさない

### スラッシュコマンドはどこでもOK

スラッシュコマンドは、どのチャンネルからでも実行できます。

- 個人チャンネルから: `/notebook_list`
- 一般チャンネルから: `/notebook_list`
- どちらでも同じように動作します

---

## 📝 まとめ

### 単語帳機能のテスト方法

1. **個人チャンネルを探す**
   - 「Winglish｜個人学習」カテゴリの `winglish-{あなたの名前}` チャンネル

2. **そのチャンネルでコマンドを実行**
   ```
   /notebook_create name:"テスト単語帳"
   /notebook_list
   /notebook_add notebook_name:"テスト単語帳" word:abandon
   /notebook_study notebook_name:"テスト単語帳"
   ```

3. **または、どのチャンネルからでも実行可能**
   - スラッシュコマンドはどこからでも使える

---

**結論**: 個人チャンネルで使うのが推奨ですが、スラッシュコマンドはどのチャンネルからでも実行できます！
